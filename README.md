# avalia-o-de-laborat-rio-de-eletromagnetismo
Projeto de desenvolvimento de simulador de Guias de Onda, Cavidade ressonante e Antena com Fendas, para disciplina de Laboratório de Eletromagnetismo do curso de Engenharia Elétrica
-> Para a Criação dos simuladores a plataforma Visual Code foi escolhida em questão. Enquanto que a linguagem Python foi requisitada como obrigatória para desenvolvimento de cada protótipo.
O relatório da experiência, bem como os dados relativos ao que foi desenvolvido deverá ser postado eventualmente após ajustes técnicos finais.
